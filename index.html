My Latest Projects

A Computational Model of a Human Neuron

A mathematical model implemented in MATLAB that can simulate the activity of a neuron, such as excitation, inhibition, and action potentials.

A Network of Simulated Neuronal Circuits

A complex set of differential equations that describe the behavior of multiple neurons in a circuit, including inherent 'stochasticity' (a random chance that a neuron will fire).

A GUI to Implement and Administer Mental Stressors Tests for a Human Subject Study

A Graphical User Interface (GUI) that can administer 3 separate mental stressor tasks and accept responses using speech recognition via the Google Speech API.
